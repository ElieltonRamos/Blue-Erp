💡 Como ler este relatório

Cada local de venda representa onde o pedido foi criado (mesa ou delivery). O local de produção é o campo Locais de produção cadastrado no produto — indica onde aquele item é preparado. Produtos sem local de produção cadastrado assumem o próprio local de venda.

O Tempo de Comanda é calculado entre a criação e o fechamento do pedido (criado → finalizado). O Tempo de Mesa usa criado → tempo de ocupação — apenas pedidos com mesa e esse campo preenchido entram no cálculo.
